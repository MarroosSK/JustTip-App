# JustTip - Tip Calculator

![Project Screenshot](https://i.postimg.cc/nrgPPwjM/tip.png)

## Description
In case you wish to tip in restaurant, this app helps you figure out amount and %.

## Tech Stack
- Expo / React Native
- Zustand
- NativeWind
- Lottie

## Features
- Gamification with badges
- History of tips
- Dark mode
- Multiple language selection
